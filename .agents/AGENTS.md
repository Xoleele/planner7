# Reglas del Workspace (Planner7)

## Navegador (Barra Superior)
- El término **Navegador** (con N mayúscula) se refiere a la barra superior de la aplicación.
- Contiene los siguientes componentes:
  - Botón para alternar modos (Lista de tareas / Línea de tiempo)
  - Botones/flechas para cambiar de semana o día (prev-week-btn, next-week-btn)
  - Rango de fechas o fecha actual (week-range-label)
  - Botón "Hoy" (today-btn)
- En el código, la clase CSS principal de esta barra es `.navegador` (anteriormente `.week-navigator`).
